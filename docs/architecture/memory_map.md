## 1. 位址空間概覽

FormosaSoC 採用 32 位元位址空間，以記憶體映射 I/O（MMIO）方式存取所有周邊暫存器。

### 主要區段表

| 起始位址 | 結束位址 | 大小 | 區段名稱 | 描述 |
|----------|----------|------|----------|------|
| `0x0000_0000` | `0x0003_FFFF` | 256 KB | ROM | 韌體程式碼儲存 |
| `0x1000_0000` | `0x1000_FFFF` | 64 KB | RAM | 資料與堆疊記憶體 |
| `0x2000_0000` | `0x2000_2FFF` | 12 KB | 系統控制 | SYSCTRL / CLKCTRL / RSTCTRL |
| `0x2001_0000` | `0x2001_3FFF` | 16 KB | PLIC | 平台級中斷控制器 |
| `0x2010_0000` | `0x20FF_FFFF` | ~15 MB | 通用周邊 | GPIO, UART, SPI, I2C 等 |
| `0x3000_0000` | `0x302F_FFFF` | 3 MB | 無線周邊 | Wi-Fi, BLE, RF |

## 2. ROM 區段

| 項目 | 值 |
|------|-----|
| 基底位址 | `0x0000_0000` |
| 大小 | 256 KB (`0x0004_0000`) |
| 結束位址 | `0x0003_FFFF` |
| 存取權限 | 唯讀（執行時） |
| 用途 | 存放 Boot Loader 及韌體程式碼 |

ROM 區段為 CPU 的啟動位址（Reset Vector），上電後 CPU 從 `0x0000_0000` 開始執行。在 FPGA 原型中，此區段映射至 FPGA 的 Block RAM，並在建構時由 LiteX 工具鏈自動填入韌體。

## 3. RAM 區段

| 項目 | 值 |
|------|-----|
| 基底位址 | `0x1000_0000` |
| 大小 | 64 KB (`0x0001_0000`) |
| 結束位址 | `0x1000_FFFF` |
| 存取權限 | 讀寫 |
| 用途 | 全域變數、BSS 區段、堆疊、堆積 |

### RAM 佈局

```
0x1000_0000  ┌────────────────────┐  ← RAM 起始
             │   .data 區段       │  初始化全域變數
             ├────────────────────┤
             │   .bss 區段        │  未初始化全域變數（清零）
             ├────────────────────┤
             │                    │
             │   堆積 (Heap) ↓    │  動態記憶體分配（向高位址成長）
             │                    │
             │        ...         │
             │                    │
             │   堆疊 (Stack) ↑   │  函式呼叫堆疊（向低位址成長）
             │                    │
0x1000_FFFF  └────────────────────┘  ← SP 初始值
```

## 4. 系統控制周邊

### 4.1 系統控制器 (SYSCTRL) — 基底位址 `0x2000_0000`

| 偏移 | 暫存器名稱 | 存取 | 描述 |
|------|-----------|------|------|
| `0x00` | CHIP_ID | RO | 晶片識別碼（固定值 `0x464D5341` = "FMSA"） |
| `0x04` | CHIP_VER | RO | 晶片版本號 |
| `0x08` | SYS_CFG | RW | 系統組態暫存器 |
| `0x0C` | PWR_CTRL | RW | 電源控制暫存器 |
| `0x10` | PWR_STATUS | RO | 電源狀態暫存器 |
| `0x14` | BOOT_MODE | RW | 開機模式暫存器 |

#### PWR_CTRL 暫存器位元欄位

| 位元 | 名稱 | 描述 |
|------|------|------|
| [0] | SLEEP | 寫 1 進入睡眠模式 |
| [1] | DEEP_SLEEP | 寫 1 進入深度睡眠模式 |
| [2] | HIBERNATE | 寫 1 進入休眠模式 |
| [31:3] | - | 保留 |

### 4.2 時脈控制器 (CLKCTRL) — 基底位址 `0x2000_1000`

| 偏移 | 暫存器名稱 | 存取 | 描述 |
|------|-----------|------|------|
| `0x00` | PLL_CFG | RW | PLL 組態暫存器 |
| `0x04` | PLL_STATUS | RO | PLL 狀態暫存器 |
| `0x08` | CLK_DIV | RW | 時脈分頻暫存器 |
| `0x0C` | CLK_EN | RW | 周邊時脈致能暫存器 |
| `0x10` | CLK_SEL | RW | 時脈來源選擇暫存器 |

#### PLL_CFG 暫存器位元欄位

| 位元 | 名稱 | 描述 |
|------|------|------|
| [5:0] | PLL_MULT | PLL 倍頻係數 |
| [7:6] | - | 保留 |
| [11:8] | PLL_DIV | PLL 分頻係數 |
| [15:12] | - | 保留 |
| [16] | PLL_EN | PLL 致能（1=啟用） |
| [31:17] | - | 保留 |

PLL 輸出頻率計算公式：

```
F_out = F_xtal * PLL_MULT / PLL_DIV
```

#### PLL_STATUS 暫存器位元欄位

| 位元 | 名稱 | 描述 |
|------|------|------|
| [0] | PLL_LOCK | PLL 鎖定指示（1=已鎖定） |
| [31:1] | - | 保留 |

#### CLK_EN 暫存器位元欄位

| 位元 | 名稱 | 描述 |
|------|------|------|
| [0] | GPIO_EN | GPIO 時脈致能 |
| [1] | UART0_EN | UART0 時脈致能 |
| [2] | UART1_EN | UART1 時脈致能 |
| [3] | SPI0_EN | SPI0 時脈致能 |
| [4] | SPI1_EN | SPI1 時脈致能 |
| [5] | I2C0_EN | I2C0 時脈致能 |
| [6] | I2C1_EN | I2C1 時脈致能 |
| [7] | PWM_EN | PWM 時脈致能 |
| [8] | TIMER_EN | Timer 時脈致能 |
| [9] | WDT_EN | WDT 時脈致能 |
| [10] | ADC_EN | ADC 時脈致能 |
| [11] | DMA_EN | DMA 時脈致能 |
| [15:12] | - | 保留 |
| [16] | WIFI_EN | Wi-Fi 時脈致能 |
| [17] | BLE_EN | BLE 時脈致能 |
| [31:18] | - | 保留 |

### 4.3 重設控制器 (RSTCTRL) — 基底位址 `0x2000_2000`

| 偏移 | 暫存器名稱 | 存取 | 描述 |
|------|-----------|------|------|
| `0x00` | SW_RST | WO | 軟體重設暫存器（寫入 `0x5A5A0001` 觸發） |
| `0x04` | PERIPH_RST | RW | 周邊裝置重設暫存器（寫 1 觸發，自動清除） |
| `0x08` | RST_STATUS | RO | 重設原因狀態暫存器 |

### 4.4 PLIC — 基底位址 `0x2001_0000`

| 偏移 | 暫存器名稱 | 存取 | 描述 |
|------|-----------|------|------|
| `0x0000`~`0x00FC` | PRIORITY[0..63] | RW | 中斷優先權暫存器陣列（每個 4 bytes） |
| `0x1000` | PENDING[0] | RO | 中斷等待暫存器（位元 0~31） |
| `0x1004` | PENDING[1] | RO | 中斷等待暫存器（位元 32~63） |
| `0x2000` | ENABLE[0] | RW | 中斷致能暫存器（位元 0~31） |
| `0x2004` | ENABLE[1] | RW | 中斷致能暫存器（位元 32~63） |
| `0x3000` | THRESHOLD | RW | 優先權閾值暫存器 |
| `0x3004` | CLAIM/COMPLETE | RW | 中斷領取/完成暫存器 |

## 7. 各周邊暫存器映射

### 7.1 GPIO (基底: `0x2010_0000`)

| 偏移 | 暫存器 | 存取 | 描述 |
|------|--------|------|------|
| `0x00` | DIR | RW | 方向暫存器（0=輸入, 1=輸出） |
| `0x04` | OUTPUT | RW | 輸出資料暫存器 |
| `0x08` | INPUT | RO | 輸入資料暫存器 |
| `0x0C` | OUTPUT_SET | WO | 輸出設定暫存器（寫 1 設定） |
| `0x10` | OUTPUT_CLR | WO | 輸出清除暫存器（寫 1 清除） |
| `0x14` | OUTPUT_TOGGLE | WO | 輸出翻轉暫存器（寫 1 翻轉） |
| `0x18` | INT_EN | RW | 中斷致能暫存器 |
| `0x1C` | INT_TYPE | RW | 中斷類型（0=準位, 1=邊緣） |
| `0x20` | INT_POL | RW | 中斷極性（0=低/下降, 1=高/上升） |
| `0x24` | INT_BOTH | RW | 雙緣觸發致能暫存器 |
| `0x28` | INT_STATUS | RO | 中斷狀態暫存器 |
| `0x2C` | INT_CLR | WO | 中斷清除暫存器（寫 1 清除） |
| `0x30` | PULL_EN | RW | 上下拉電阻致能暫存器 |
| `0x34` | PULL_SEL | RW | 上下拉選擇（0=下拉, 1=上拉） |
| `0x38` | ALT_FUNC | RW | 替代功能選擇暫存器 |

### 7.2 UART (基底: `0x2020_0000` / `0x2020_1000`)

| 偏移 | 暫存器 | 存取 | 描述 |
|------|--------|------|------|
| `0x00` | DATA | RW | 資料暫存器（讀=接收, 寫=傳送） |
| `0x04` | STATUS | RO | 狀態暫存器 |
| `0x08` | CTRL | RW | 控制暫存器 |
| `0x0C` | BAUD_DIV | RW | 鮑率除數暫存器 |
| `0x10` | INT_EN | RW | 中斷致能暫存器 |
| `0x14` | INT_STATUS | RO | 中斷狀態暫存器 |
| `0x18` | INT_CLR | WO | 中斷清除暫存器 |
| `0x1C` | FIFO_STATUS | RO | FIFO 狀態暫存器 |

#### STATUS 暫存器位元欄位

| 位元 | 名稱 | 描述 |
|------|------|------|
| [0] | TX_FULL | 傳送 FIFO 滿 |
| [1] | RX_EMPTY | 接收 FIFO 空 |
| [2] | TX_EMPTY | 傳送 FIFO 空 |
| [3] | RX_FULL | 接收 FIFO 滿 |
| [4] | OVERRUN | 溢位錯誤 |
| [5] | FRAME_ERR | 框架錯誤 |
| [6] | PARITY_ERR | 奇偶校驗錯誤 |

#### CTRL 暫存器位元欄位

| 位元 | 名稱 | 描述 |
|------|------|------|
| [0] | TX_EN | 傳送致能 |
| [1] | RX_EN | 接收致能 |
| [2] | PARITY_EN | 奇偶校驗致能 |
| [3] | PARITY_SEL | 校驗模式（0=偶, 1=奇） |
| [4] | STOP_BITS | 停止位元（0=1bit, 1=2bit） |
| [5] | FIFO_EN | FIFO 致能 |

#### FIFO_STATUS 暫存器位元欄位

| 位元 | 名稱 | 描述 |
|------|------|------|
| [4:0] | TX_COUNT | 傳送 FIFO 中的位元組數 |
| [12:8] | RX_COUNT | 接收 FIFO 中的位元組數 |

| 位元 | 名稱 | 描述 |

| [8] | ONESHOT | 單次觸發模式 |

### 7.7 WDT (基底: `0x2070_0000`)

| 偏移 | 暫存器 | 存取 | 描述 |
|------|--------|------|------|
| `0x00` | LOAD | RW | 看門狗載入值暫存器 |
| `0x04` | VALUE | RO | 看門狗當前值暫存器 |
| `0x08` | CTRL | RW | 看門狗控制暫存器 |
| `0x0C` | INT_CLR | WO | 看門狗中斷清除暫存器 |
| `0x10` | LOCK | RW | 看門狗鎖定暫存器 |
| `0x14` | INT_STATUS | RO | 看門狗中斷狀態暫存器 |

#### CTRL 暫存器位元欄位

| 位元 | 名稱 | 描述 |
|------|------|------|
| [0] | EN | 看門狗致能 |
| [1] | RST_EN | 重設致能（逾時時觸發系統重設） |
| [2] | INT_EN | 中斷致能（逾時前先觸發中斷警告） |

#### LOCK 暫存器

- 寫入 `0x1ACCE551` 解鎖看門狗暫存器
- 寫入任何其他值鎖定暫存器（防止誤修改）
