# FormosaSoC GPIO 控制器資料手冊

**文件版本：** 1.0
**日期：** 2026-03-03
**作者：** FormosaSoC 開發團隊

---

## 1. 功能特色

- 32 支通用輸入/輸出腳位
- 每支腳位可獨立設定為輸入或輸出
- 硬體原子操作：設定 (SET)、清除 (CLR)、翻轉 (TOGGLE)
- 多種中斷觸發模式：上升緣、下降緣、雙緣、高準位、低準位
- 內建可程式化上拉/下拉電阻
- 替代功能複用（周邊腳位映射）
- Wishbone B4 從端介面
- 每腳位獨立中斷致能

## 3. 暫存器映射

基底位址：`0x2010_0000`

| 偏移 | 暫存器名稱 | 存取 | 復位值 | 描述 |
|------|-----------|------|--------|------|
| `0x00` | DIR | RW | `0x0000_0000` | 方向暫存器 |
| `0x04` | OUTPUT | RW | `0x0000_0000` | 輸出資料暫存器 |
| `0x08` | INPUT | RO | - | 輸入資料暫存器 |
| `0x0C` | OUTPUT_SET | WO | - | 輸出設定暫存器 |
| `0x10` | OUTPUT_CLR | WO | - | 輸出清除暫存器 |
| `0x14` | OUTPUT_TOGGLE | WO | - | 輸出翻轉暫存器 |
| `0x18` | INT_EN | RW | `0x0000_0000` | 中斷致能暫存器 |
| `0x1C` | INT_TYPE | RW | `0x0000_0000` | 中斷類型暫存器 |
| `0x20` | INT_POL | RW | `0x0000_0000` | 中斷極性暫存器 |
| `0x24` | INT_BOTH | RW | `0x0000_0000` | 雙緣觸發致能暫存器 |
| `0x28` | INT_STATUS | RO | `0x0000_0000` | 中斷狀態暫存器 |
| `0x2C` | INT_CLR | WO | - | 中斷清除暫存器 |
| `0x30` | PULL_EN | RW | `0x0000_0000` | 上下拉電阻致能暫存器 |
| `0x34` | PULL_SEL | RW | `0x0000_0000` | 上下拉選擇暫存器 |
| `0x38` | ALT_FUNC | RW | `0x0000_0000` | 替代功能選擇暫存器 |

## 4. 暫存器詳細說明

### 4.1 DIR — 方向暫存器 (偏移 `0x00`)

每個位元控制對應腳位的方向。

| 位元 | 值 | 描述 |
|------|-----|------|
| [n] | 0 | GPIO[n] 為輸入模式 |
| [n] | 1 | GPIO[n] 為輸出模式 |

### 4.2 OUTPUT — 輸出資料暫存器 (偏移 `0x04`)

直接設定所有腳位的輸出值。讀取時回傳當前輸出鎖存值。

### 4.3 INPUT — 輸入資料暫存器 (偏移 `0x08`)

讀取所有腳位的實際邏輯準位（不論方向設定）。

### 4.4 OUTPUT_SET — 輸出設定暫存器 (偏移 `0x0C`)

寫入 1 將對應位元設定為高準位，寫入 0 不影響。此暫存器為原子操作，無讀取-修改-寫入的競爭條件。

### 4.5 OUTPUT_CLR — 輸出清除暫存器 (偏移 `0x10`)

寫入 1 將對應位元清除為低準位，寫入 0 不影響。

### 4.6 OUTPUT_TOGGLE — 輸出翻轉暫存器 (偏移 `0x14`)

寫入 1 翻轉對應位元的輸出值，寫入 0 不影響。

### 4.7 中斷配置

中斷觸發模式由 INT_TYPE、INT_POL、INT_BOTH 三個暫存器組合決定：

| INT_BOTH[n] | INT_TYPE[n] | INT_POL[n] | 觸發模式 |
|-------------|-------------|------------|----------|
| 0 | 0 | 0 | 低準位觸發 |
| 0 | 0 | 1 | 高準位觸發 |
| 0 | 1 | 0 | 下降緣觸發 |
| 0 | 1 | 1 | 上升緣觸發 |
| 1 | 1 | X | 雙緣觸發（上升+下降） |

### 4.8 PULL_EN / PULL_SEL — 上下拉電阻配置

| PULL_EN[n] | PULL_SEL[n] | 功能 |
|------------|-------------|------|
| 0 | X | 無上下拉（浮接） |
| 1 | 0 | 下拉電阻 |
| 1 | 1 | 上拉電阻 |

## 5. 使用範例

### 5.1 基本輸出 — LED 閃爍

```c
#include "formosa_soc.h"

#define LED_PIN  0  /* GPIO[0] 連接 LED */

int main(void)
{
    /* 設定 GPIO[0] 為輸出 */
    GPIO_DIR |= (1 << LED_PIN);

    while (1) {
        /* 翻轉 LED */
        GPIO_OUTPUT_TOGGLE = (1 << LED_PIN);

        /* 簡單延遲 */
        for (volatile int i = 0; i < 1000000; i++);
    }
}
```

### 5.2 基本輸入 — 讀取按鍵

```c
#include "formosa_soc.h"

#define BTN_PIN  1  /* GPIO[1] 連接按鍵 */

int main(void)
{
    /* 設定 GPIO[1] 為輸入 */
    GPIO_DIR &= ~(1 << BTN_PIN);

    /* 啟用上拉電阻 */
    GPIO_PULL_EN  |= (1 << BTN_PIN);
    GPIO_PULL_SEL |= (1 << BTN_PIN);

    while (1) {
        if (!(GPIO_INPUT & (1 << BTN_PIN))) {
            /* 按鍵按下（低電位） */
        }
    }
}
```

### 5.3 中斷驅動 — 按鍵偵測

```c
#include "formosa_soc.h"
#include "gpio.h"

#define BTN_PIN  1

void button_callback(uint32_t pin)
{
    if (pin == BTN_PIN) {
        GPIO_OUTPUT_TOGGLE = (1 << 0);  /* 翻轉 LED */
    }
}

int main(void)
{
    gpio_init();

    /* 設定按鍵為輸入，上拉 */
    gpio_set_dir(BTN_PIN, GPIO_DIR_INPUT);
    gpio_set_pull(BTN_PIN, GPIO_PULL_UP);

    /* 設定下降緣中斷 */
    gpio_set_interrupt(BTN_PIN, GPIO_IRQ_FALLING, button_callback);

    /* 設定 LED 為輸出 */
    gpio_set_dir(0, GPIO_DIR_OUTPUT);

    /* 致能全域中斷 */
    formosa_enable_interrupts();

    while (1) {
        /* 主迴圈可處理其他事務 */
    }
}
```

### 5.4 使用驅動程式 API

```c
#include "gpio.h"

int main(void)
{
    gpio_init();

    /* 設定方向 */
    gpio_set_dir(0, GPIO_DIR_OUTPUT);   /* Pin 0 輸出 */
    gpio_set_dir(1, GPIO_DIR_INPUT);    /* Pin 1 輸入 */

    /* 設定上下拉 */
    gpio_set_pull(1, GPIO_PULL_UP);

    /* 寫入/讀取 */
    gpio_write(0, 1);                   /* Pin 0 拉高 */
    int val = gpio_read(1);             /* 讀取 Pin 1 */

    /* 翻轉 */
    gpio_toggle(0);                     /* Pin 0 翻轉 */

    return 0;
}
```
