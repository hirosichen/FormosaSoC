# FormosaSoC UART 控制器資料手冊

**文件版本：** 1.0
**日期：** 2026-03-03
**作者：** FormosaSoC 開發團隊

---

## 1. 功能特色

- 2 組獨立 UART 控制器（UART0, UART1）
- 可程式化鮑率（最高支援 APB_CLK / 16）
- 16 位元組深度 TX/RX FIFO 緩衝
- 支援 8 位元資料格式
- 可選奇偶校驗（偶校驗/奇校驗）
- 可選 1 或 2 個停止位元
- 錯誤偵測：溢位、框架錯誤、奇偶校驗錯誤
- 3 個中斷源：TX 空、RX 就緒、RX 溢位
- Wishbone B4 從端介面

## 3. 暫存器映射

基底位址：UART0 = `0x2020_0000`，UART1 = `0x2020_1000`

| 偏移 | 暫存器名稱 | 存取 | 復位值 | 描述 |
|------|-----------|------|--------|------|
| `0x00` | DATA | RW | `0x0000_0000` | 資料暫存器（讀=接收, 寫=傳送） |
| `0x04` | STATUS | RO | `0x0000_0006` | 狀態暫存器 |
| `0x08` | CTRL | RW | `0x0000_0000` | 控制暫存器 |
| `0x0C` | BAUD_DIV | RW | `0x0000_0000` | 鮑率除數暫存器 |
| `0x10` | INT_EN | RW | `0x0000_0000` | 中斷致能暫存器 |
| `0x14` | INT_STATUS | RO | `0x0000_0000` | 中斷狀態暫存器 |
| `0x18` | INT_CLR | WO | - | 中斷清除暫存器 |
| `0x1C` | FIFO_STATUS | RO | `0x0000_0000` | FIFO 狀態暫存器 |

## 4. 暫存器詳細說明

### 4.1 DATA — 資料暫存器 (偏移 `0x00`)

- **寫入**：將字元寫入 TX FIFO。若 TX FIFO 已滿，寫入將被忽略。
- **讀取**：從 RX FIFO 讀取一個字元。若 RX FIFO 為空，讀取值無效。

| 位元 | 名稱 | 描述 |
|------|------|------|
| [7:0] | DATA | 資料位元組 |
| [31:8] | - | 保留 |

### 4.2 STATUS — 狀態暫存器 (偏移 `0x04`)

| 位元 | 名稱 | 描述 |
|------|------|------|
| [0] | TX_FULL | 傳送 FIFO 已滿（1=滿） |
| [1] | RX_EMPTY | 接收 FIFO 為空（1=空） |
| [2] | TX_EMPTY | 傳送 FIFO 為空（1=空） |
| [3] | RX_FULL | 接收 FIFO 已滿（1=滿） |
| [4] | OVERRUN | 接收溢位錯誤（RX FIFO 滿時仍收到資料） |
| [5] | FRAME_ERR | 框架錯誤（未偵測到正確的停止位元） |
| [6] | PARITY_ERR | 奇偶校驗錯誤 |
| [31:7] | - | 保留 |

### 4.3 CTRL — 控制暫存器 (偏移 `0x08`)

| 位元 | 名稱 | 描述 |
|------|------|------|
| [0] | TX_EN | 傳送致能（1=啟用） |
| [1] | RX_EN | 接收致能（1=啟用） |
| [2] | PARITY_EN | 奇偶校驗致能（1=啟用） |
| [3] | PARITY_SEL | 校驗模式選擇（0=偶校驗, 1=奇校驗） |
| [4] | STOP_BITS | 停止位元數（0=1 個, 1=2 個） |
| [5] | FIFO_EN | FIFO 致能（1=啟用） |
| [31:6] | - | 保留 |

### 4.4 BAUD_DIV — 鮑率除數暫存器 (偏移 `0x0C`)

鮑率除數計算公式：

```
divisor = APB_CLOCK / (16 * baud_rate)
```

以預設 APB 時脈 40 MHz 為例：

| 目標鮑率 | 除數 (divisor) | 實際鮑率 | 誤差 |
|----------|---------------|---------|------|
| 9600 | 260 | 9615 | +0.16% |
| 19200 | 130 | 19231 | +0.16% |
| 38400 | 65 | 38462 | +0.16% |
| 57600 | 43 | 58140 | +0.94% |
| 115200 | 21 | 119048 | +3.34% |
| 230400 | 10 | 250000 | +8.51% |
| 460800 | 5 | 500000 | +8.51% |
| 921600 | 2 | 1250000 | +35.6% |

> **注意：** 高鮑率時誤差較大，建議鮑率不超過 460800 baud。

### 4.5 INT_EN — 中斷致能暫存器 (偏移 `0x10`)

| 位元 | 名稱 | 描述 |
|------|------|------|
| [0] | TX_EMPTY_IE | TX FIFO 空中斷致能 |
| [1] | RX_READY_IE | RX 資料就緒中斷致能 |
| [2] | RX_OVERRUN_IE | RX 溢位中斷致能 |
| [31:3] | - | 保留 |

### 4.6 FIFO_STATUS — FIFO 狀態暫存器 (偏移 `0x1C`)

| 位元 | 名稱 | 描述 |
|------|------|------|
| [4:0] | TX_COUNT | TX FIFO 中的位元組數（0~16） |
| [7:5] | - | 保留 |
| [12:8] | RX_COUNT | RX FIFO 中的位元組數（0~16） |
| [31:13] | - | 保留 |

## 5. 通訊格式

```
UART 資料幀格式：

         ┌────┬───┬───┬───┬───┬───┬───┬───┬───┬──────┬────┐
閒置(高) │起始│D0 │D1 │D2 │D3 │D4 │D5 │D6 │D7 │校驗位│停止│ 閒置(高)
         │位元│   │   │   │   │   │   │   │   │(選用)│位元│
         └────┴───┴───┴───┴───┴───┴───┴───┴───┴──────┴────┘
           ↑                                          ↑
        下降緣                                     上升緣
        (低電位)                                   (高電位)

位元時序：每個位元持續 1/baud_rate 秒
```

## 6. 使用範例

### 6.1 基本初始化與字元傳送

```c
#include "formosa_soc.h"

void uart_basic_init(void)
{
    /* 設定鮑率 115200 (APB_CLK=40MHz) */
    UART_BAUD_DIV(FORMOSA_UART0_BASE) = 40000000 / (16 * 115200);

    /* 致能 TX/RX，啟用 FIFO */
    UART_CTRL(FORMOSA_UART0_BASE) = UART_CTRL_TX_EN_Msk |
                                     UART_CTRL_RX_EN_Msk |
                                     UART_CTRL_FIFO_EN_Msk;
}

void uart_send_char(char ch)
{
    /* 等待 TX FIFO 非滿 */
    while (UART_STATUS(FORMOSA_UART0_BASE) & UART_STATUS_TX_FULL_Msk);

    /* 寫入字元 */
    UART_DATA(FORMOSA_UART0_BASE) = ch;
}

char uart_recv_char(void)
{
    /* 等待 RX FIFO 非空 */
    while (UART_STATUS(FORMOSA_UART0_BASE) & UART_STATUS_RX_EMPTY_Msk);

    /* 讀取字元 */
    return (char)UART_DATA(FORMOSA_UART0_BASE);
}
```

### 6.2 使用驅動程式 API

```c
#include "uart.h"

int main(void)
{
    /* 使用預設組態初始化 UART0 */
    uart_config_t cfg = UART0_DEFAULT_CONFIG;
    uart_init(&cfg);

    /* 傳送字串 */
    uart_puts(FORMOSA_UART0_BASE, "FormosaSoC 啟動成功！\n");

    /* 格式化輸出 */
    uart_printf(FORMOSA_UART0_BASE, "系統時脈: %d MHz\n",
                FORMOSA_SYSTEM_CLOCK_HZ / 1000000);

    /* 回音測試 */
    while (1) {
        char ch = uart_getc(FORMOSA_UART0_BASE);
        uart_putc(FORMOSA_UART0_BASE, ch);
    }
}
```

### 6.3 奇偶校驗設定

```c
uart_config_t cfg = {
    .base_addr  = FORMOSA_UART0_BASE,
    .baud_rate  = 9600,
    .parity_en  = 1,      /* 啟用校驗 */
    .parity_odd = 0,      /* 偶校驗 */
    .stop_bits  = 0,      /* 1 個停止位元 */
    .fifo_en    = 1
};
uart_init(&cfg);
```

### 6.4 查詢可讀取資料量

```c
/* 非阻塞式檢查 */
if (uart_available(FORMOSA_UART0_BASE) > 0) {
    char ch = uart_getc(FORMOSA_UART0_BASE);
    /* 處理接收到的字元 */
}
```
