## 2. Tang Nano 20K е»әж§ӢжҢҮеҚ—

### 2.1 й–Ӣзҷјз’°еўғиЁӯзҪ®

#### ж–№жі•дёҖпјҡGowin IDEпјҲе®ҳж–№е·Ҙе…·пјҢжҺЁи–Ұпјү

1. еүҚеҫҖй«ҳйӣІеҚҠе°Һй«”е®ҳз¶ІиЁ»еҶҠеёіиҷҹ
2. дёӢијү Gowin EDAпјҲж•ҷиӮІзүҲе…ҚиІ»пјү
3. з”іи«Ӣе…ҚиІ»жҺҲж¬ҠжӘ”жЎҲ
4. е®үиЈқдёҰе•ҹз”Ё

```bash
# е®үиЈқеҫҢзўәиӘҚе·Ҙе…·йҸҲи·Ҝеҫ‘
which gw_sh
# жҲ–еҠ е…Ҙ PATH
export PATH=/opt/gowin/IDE/bin:$PATH
```

#### ж–№жі•дәҢпјҡApiculaпјҲй–Ӣжәҗе·Ҙе…·йҸҲпјү

```bash
# е®үиЈқ Apicula
pip install apicula

# е®үиЈқ YosysпјҲеҗҲжҲҗеҷЁпјү
sudo apt install yosys

# е®үиЈқ nextpnr-gowinпјҲдҪҲеұҖдҪҲз·ҡпјү
# йңҖеҫһеҺҹе§Ӣзўјз·ЁиӯҜ
git clone https://github.com/YosysHQ/nextpnr.git
cd nextpnr
cmake -DARCH=gowin .
make -j$(nproc)
sudo make install
```

## 3. Arty A7-35T е»әж§ӢжҢҮеҚ—

### 3.1 й–Ӣзҷјз’°еўғиЁӯзҪ®

#### е®үиЈқ Vivado

1. еүҚеҫҖ AMD/Xilinx е®ҳз¶ІдёӢијү Vivado ML Edition
2. йҒёж“Үе…ҚиІ»зҡ„ Standard Edition
3. е®үиЈқжҷӮеғ…йҒёж“Ү "Artix-7" иЈқзҪ®ж”ҜжҸҙпјҲжёӣе°‘е®үиЈқеӨ§е°Ҹпјү
4. е®үиЈқе®ҢжҲҗеҫҢиЁӯе®ҡз’°еўғпјҡ

```bash
# иЁӯе®ҡ Vivado з’°еўғпјҲLinuxпјү
source /tools/Xilinx/Vivado/2024.1/settings64.sh

# зўәиӘҚе®үиЈқ
vivado -version
```

### 3.2 е»әж§ӢжӯҘй©ҹ

```bash
cd FormosaSoC/soc

# жӯҘй©ҹ 1пјҡеғ…з”ўз”ҹ Verilog RTL
python formosa_soc.py --target arty_a7 --no-compile-gateware

# жӯҘй©ҹ 2пјҡе®Ңж•ҙе»әж§Ӣ
python formosa_soc.py --target arty_a7 --build
# жіЁж„ҸпјҡVivado еҗҲжҲҗеҸҜиғҪйңҖиҰҒ 5-15 еҲҶйҗҳ

# жӯҘй©ҹ 3пјҡжҢҮе®ҡзі»зөұжҷӮи„Ҳе»әж§Ӣ
python formosa_soc.py --target arty_a7 --sys-clk-freq 100e6 --build

# жӯҘй©ҹ 4пјҡијүе…Ҙ FPGA
python formosa_soc.py --target arty_a7 --build --load

# жӯҘй©ҹ 5пјҡе•ҹз”Ё CPU йҷӨйҢҜд»Ӣйқў
python formosa_soc.py --target arty_a7 --with-debug --build
```

## 6. зҮ’йҢ„иҲҮијүе…Ҙ

### 6.1 Tang Nano 20K зҮ’йҢ„

**дҪҝз”Ё LiteXпјҲжҺЁи–Ұпјүпјҡ**

```bash
python formosa_soc.py --target tang_nano_20k --build --load
```

**дҪҝз”Ё Gowin Programmerпјҡ**

1. й–Ӣе•ҹ Gowin Programmer
2. йҒёж“Ү "SRAM Program"пјҲжҸ®зҷјжҖ§ијүе…ҘпјүжҲ– "FLASH Program"пјҲж°ёд№…зҮ’йҢ„пјү
3. йҒёж“Ү `.bit` жҲ– `.fs` жӘ”жЎҲ
4. й»һж“Ҡ "Program/Configure"

### 6.2 Arty A7 зҮ’йҢ„

**дҪҝз”Ё LiteXпјҡ**

```bash
python formosa_soc.py --target arty_a7 --build --load
```

**дҪҝз”Ё Vivado Hardware Managerпјҡ**

1. й–Ӣе•ҹ Vivado
2. йҒёж“Ү "Open Hardware Manager"
3. й»һж“Ҡ "Open Target" вҶ’ "Auto Connect"
4. еҸійҚө FPGA иЈқзҪ® вҶ’ "Program Device"
5. йҒёж“Ү `.bit` жӘ”жЎҲ
6. й»һж“Ҡ "Program"

**дҪҝз”Ёе‘Ҫд»ӨеҲ—пјҡ**

```bash
# дҪҝз”Ё Vivado е‘Ҫд»ӨеҲ—зҮ’йҢ„
vivado -mode batch -source program.tcl

# program.tcl е…§е®№пјҡ
# open_hw_manager
# connect_hw_server
# open_hw_target
# set_property PROGRAM.FILE {build/arty_a7/gateware/arty_a7.bit} [current_hw_device]
# program_hw_devices [current_hw_device]
# close_hw_target
# disconnect_hw_server
# close_hw_manager
```

## 8. еёёиҰӢе•ҸйЎҢ

### Q1пјҡеҗҲжҲҗжҷӮеҮәзҸҫ "timing violation" йҢҜиӘӨ

**еҺҹеӣ пјҡ** зі»зөұжҷӮи„Ҳй »зҺҮйҒҺй«ҳпјҢз„Ўжі•еңЁжҢҮе®ҡжҷӮй–“е…§е®ҢжҲҗжүҖжңүйӮҸијҜи·Ҝеҫ‘гҖӮ

**и§Јжұәж–№ејҸпјҡ**
- йҷҚдҪҺзі»зөұжҷӮи„Ҳй »зҺҮпјҲеҰӮ `--sys-clk-freq 50e6`пјү
- з°ЎеҢ–иЁӯиЁҲпјҲеҒңз”ЁдёҚйңҖиҰҒзҡ„е‘ЁйӮҠпјү
- жӘўжҹҘй—ңйҚөи·Ҝеҫ‘дёҰжңҖдҪіеҢ–

### Q2пјҡFPGA иіҮжәҗдёҚи¶і (Resource Utilization > 100%)

**и§Јжұәж–№ејҸпјҡ**
- еҒңз”ЁдёҚйңҖиҰҒзҡ„е‘ЁйӮҠпјҲ`--no-pwm`, `--no-watchdog` зӯүпјү
- жёӣе°‘ SRAM/ROM еӨ§е°Ҹ
- еҚҮзҙҡеҲ°жӣҙеӨ§зҡ„ FPGAпјҲеҰӮ Arty A7-100Tпјү

### Q3пјҡзҮ’йҢ„еҫҢ UART з„ЎијёеҮә

**жҺ’жҹҘжӯҘй©ҹпјҡ**
1. зўәиӘҚ USB з·ҡйҖЈжҺҘжӯЈеёё
2. зўәиӘҚдёІеҸЈзөӮз«Ҝй®‘зҺҮиЁӯе®ҡзӮә 115200
3. зўәиӘҚдёІеҸЈиЈқзҪ®йҒёж“ҮжӯЈзўәпјҲ`/dev/ttyUSB0` жҲ– `/dev/ttyACM0`пјү
4. жҢүдёӢй–ӢзҷјжқҝйҮҚиЁӯжҢүйҚө
5. жӘўжҹҘ LED жҳҜеҗҰжңүиҝҪйҖҗзҮҲж•ҲжһңпјҲзўәиӘҚ FPGA е·Іијүе…Ҙпјү

### Q4пјҡGowin IDE жҺҲж¬Ҡе•ҸйЎҢ

**и§Јжұәж–№ејҸпјҡ**
- з”іи«Ӣ Gowin ж•ҷиӮІзүҲе…ҚиІ»жҺҲж¬Ҡ
- жҲ–дҪҝз”Ёй–Ӣжәҗ Apicula е·Ҙе…·йҸҲпјҲз„ЎйңҖжҺҲж¬Ҡпјү

### Q5пјҡVivado жүҫдёҚеҲ° FPGA иЈқзҪ®

**и§Јжұәж–№ејҸпјҡ**
- зўәиӘҚ USB з·ҡе·ІйҖЈжҺҘ
- е®үиЈқ Digilent й©…еӢ•зЁӢејҸ
- еҹ·иЎҢ `sudo apt install libftdi1-2`пјҲLinuxпјү
- жӘўжҹҘ udev иҰҸеүҮиЁӯе®ҡ
